## How to Add Your Organization

If you're using DeCube, we'd love to hear about it! To add your organization to this list:

1. Fork the repository
2. Add your organization to the appropriate section
3. Include:
   - Organization name
   - Use case (optional)
   - Link to your website or project (optional)
4. Open a pull request

Example:

```markdown
## Production Users

- **Example Corp** - Using DeCube for distributed data storage across multiple data centers
  - Website: https://example.com
  - Use case: Multi-region data replication
```
